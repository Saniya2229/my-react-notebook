
# ⚛️ React Practice Projects

This repository contains a collection of beginner to intermediate-level React projects built to improve frontend development skills using modern tools like Vite, Tailwind CSS, and JavaScript.

Each folder represents a separate project that focuses on specific concepts or features in React.

---

## 📁 Folder Overview

| Folder Name              | Description |
|--------------------------|-------------|
| `01basicreact`           | Getting started with basic React concepts like JSX, components, props, and state. |
| `01vitereact`            | A simple React app bootstrapped with Vite for faster development experience. |
| `02counter`              | A counter app demonstrating useState for incrementing and resetting values. |
| `03tailwindprops`        | React with Tailwind CSS and props passing to components. |
| `04bgChanger`            | Background color changer app using useState and random hex values. |
| `05passwordGenerator`    | Generate random passwords with customization using checkboxes and state. |
| `06currencyConverter`    | Currency conversion app using external API and user input handling. |
| `customReact`            | Custom React utilities, hooks, or layout experiments for deeper learning. |

---

## 🚀 Tech Stack

- React (with Vite)
- JavaScript (ES6+)
- Tailwind CSS
- HTML5/CSS3

---

## 🔧 How to Run

1. Clone this repository
2. Navigate into any project folder
3. Install dependencies:  
   ```bash
   npm install
   ```
4. Start the development server:  
   ```bash
   npm run dev
   ```
